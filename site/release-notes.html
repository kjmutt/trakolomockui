Release notes

What shipped in each version. Cloud gets these continuously as they ship; Standalone/On-premise installs them as discrete, versioned upgrades — see the upgrade guide before jumping several versions at once.

2026.7.2Jul 21, 2026
CloudStandalone
  • AddedDocument library — folders, files, video, audio and images side by side, with versioning and an "attach to…" flow onto tickets, KB articles, or wiki pages.
  • AddedConfluence-style wiki spaces — nested pages per team, page history, comments.
  • AddedEmployee self-service HR — joining date, designation, offer letter, payslips, leave entitlement and requests, working hours.
  • AddedNotifications and a logged-in profile menu on the Self-Service Portal and the platform console, matching the agent workspace.
  • ImprovedLicense & billing now shows a Cloud subscription view (plan, seats, invoices) alongside the on-premise license view.
  • ImprovedTenant workspaces (platform console) now shows database and file storage consumed per tenant, plus platform-wide totals.
  • FixedPrimary buttons could render invisible (white text on a light background) while hovered — a CSS specificity bug affecting every page.
  • FixedNotification and search popovers could become unclickable if another popover was already open.

CloudStandalone
  • AddedSAML SSO & SCIM provisioning — bring your own identity provider, with attribute and group→role mapping.
  • AddedCustom roles & a per-area permission matrix, alongside the built-in Admin/Manager/Agent/Viewer roles.
  • AddedWorkspace-level API keys — scoped, revocable, separate from each user's personal token.
